## Overview

This project analyzes trending Tamil artists on Spotify using real-world streaming data sourced from Kaggle.  
The objective is to transform raw streaming data into structured insights that reveal artist performance, engagement patterns, and growth trends.

---

## Business Objective

- Identify top-performing Tamil artists  
- Analyze streaming growth trends  
- Measure correlation between popularity score and stream count  
- Support data-driven music marketing decisions  

---

## Tech Stack

- SQL – Data querying and aggregation  
- Microsoft Excel – Data cleaning and preprocessing  
- Python (Pandas, NumPy, Matplotlib) – Exploratory Data Analysis  
- Power BI – Interactive dashboard development  
- Claude AI – Insight refinement and validation  

---

## Project Workflow

1. Data acquisition from Kaggle  
2. Data cleaning and validation  
3. SQL-based transformation and ranking  
4. Exploratory Data Analysis using Python  
5. Dashboard creation in Power BI  

---

## Key Insights

- Market concentration among top artists  
- Strong positive relationship between popularity score and streams  
- Engagement spikes after new releases  
- Emerging artists showing upward growth trend  

---

## Dashboard Features

- Dynamic filtering by artist and year  
- Top-N comparison analysis  
- Trend visualization  
- KPI-based performance tracking
